# 🛠️ دليل تعديل موقع Vizora

أهلاً بك! هذا الدليل سيشرح لك خطوة بخطوة كيف تقوم بتغيير كافة البيانات في الموقع (الصور، نصوص العروض، أرقام التواصل، حسابات الدفع، إلخ) لتعمل ببياناتك الحقيقية.

جميع الملفات التي ستحتاج لتعديلها موجودة داخل مجلد `src`.

---

## 1. 🖼️ تغيير صور معرض الأعمال واللاعبين
كل تفاصيل اللاعبين، الصور، وأسمائهم موجودة في ملف واحد يسهل تعديله.
- **الملف:** `src/data/mockData.ts`
- **طريقة التعديل:**
  ستجد قائمة اسمها `players`، كل لاعب له `id` و `name` و `avatarUrl` (صورته الشخصية) وقائمة `posters` (البوسترات الخاصة به).
  قم بتغيير الروابط (أمام `imageUrl`) بمسار صورك الحقيقية.
  *تأكد من رفع صورك على موقع استضافة صور (مثل Imgur) للنسخة الحية، أو وضعها في مجلد `public` واستدعائها هكذا `/image1.jpg` .*

---

## 2. 📱 تغيير أرقام التواصل (الواتساب والهاتف)
أرقام الواتساب ورابط الاتصال موجودة في عدة أماكن لأنها تؤدي وظائف مختلفة. ستحتاج للبحث عن الرقم `201000000000` (أو `01000000000`) وتغييره برقمك في الملفات التالية:

1. **الزر العائم للواتساب (تلاقيه في كل الصفحات):**
   - **الملف:** `src/components/ui/WhatsAppFab.tsx`
   - **ماذا تغير:** `https://wa.me/201000000000` بوضع رقمك الخاص.

2. **قسم الدفع والتواصل (رقم فودافون كاش والتليفون):**
   - **الملف:** `src/sections/ContactAndPayment.tsx`
   - **ماذا تغير:** 
     - رقم فودافون كاش المعروض في الشاشة `01000000000`
     - صلة الواتساب `https://wa.me/201000000000` 
     - رابط الهاتف `tel:+201000000000`

3. **نموذج الطلب (عند ملء الـ Form):**
   - **الملف:** `src/sections/OrderFlow.tsx`
   - **صاحبة المتغير اسمها:** `const WHATSAPP_NUMBER = "201000000000";` (في أعلى الدالة).

4. **أزرار باقات الأسعار (شراء باقة معينة):**
   - **الملف:** `src/sections/Pricing.tsx`
   - **ماذا تغير:** ابحث عن `https://wa.me/201000000000` وقم بتغييره.

5. **رسالة الفوز في عجلة الحظ:**
   - **الملف:** `src/sections/SpinWheel.tsx`
   - **ماذا تغير:** ابحث عن `https://wa.me/201000000000` في كود الفوز وقم بتغيير الرقم.

## 3. 🎥 قسم "قبل وبعد" (الصور التي تسحبها بالماوس)

  تحتوي كل مجموعة على `before` (رابط الصورة قبل) و `after` (رابط الصورة بعد). قم بتبديل الروابط.

## 4. 🎬 فيديوهات كواليس التصميم

  تحتوي كل مجموعة على `videoUrl` (رابط الفيديو) و `thumbnail` (صورة الغلاف قبل التشغيل). ضع روابط الفيديوهات الخاصة بك.

## 9. 🔒 إخفاء الأعمال الخاصة من الصفحة الرئيسية (لعميل مخصص)
لقد قمت ببرمجة خاصية توفر عليك الوقت، تتيح لك وضع تصميم لأحد عملائك وتبعث له الرابط (بحيث يرى ألبومه فقط) **بدون أن تظهر صوره نهائياً في الصفحة الرئيسية للموقع**.

لديك طريقتين للقيام بذلك في ملف `src/data/mockData.ts`:

### الطريقة الأولى: إخفاء ألبوم كامل لعميل (أسهل طريقة)
بدلاً من كتابة أمر الإخفاء بجوار كل بوستر، يمكنك كتابته مرة واحدة بجوار اسم اللاعب نفسه. بهذه الطريقة لن تظهر أي صورة من صوره في الرئيسية.

- **مثال:**
```typescript
{
  id: "client1",
  name: "ألبوم برايفت للعميل محمد",
  avatarUrl: "رابط-الصورة",
  hideFromHome: true, // 👈 السطر السحري هنا يخفي جميع أعمال هذا العميل بالكامل!
  posters: [
    { id: "1", title: "بوستر 1", imageUrl: "رابط", category: "مهاجم" },
    { id: "2", title: "بوستر 2", imageUrl: "رابط", category: "مهاجم" }
  ]
}
```

### الطريقة الثانية: إخفاء بوستر واحد فقط
إذا كان لديك لاعب (مثلاً كريستيانو) صوره تظهر في الرئيسية بشكل طبيعي، ولكنك صممت له "صورة خاصة" تريد إخفاءها فقط دون إخفاء باقي صوره:
- **مثال:**
```typescript
{ 
  id: "special_1", 
  title: "تصميم خاص", 
  imageUrl: "رابط-الصورة-هنا", 
  category: "مهاجم", 
  hideFromHome: true // 👈 السطر السحري هنا يخفي هذه الصورة فقط!
}
```

## 💡 نصائح هامة
1. **أبعاد الصور:** لقد قمنا بتثبيت المقاس ليكون `1059 للطول / 1488 للارتفاع`. لكي يظهر الموقع بأفضل شكل دائمًا كما اتفقنا، حاول أن تكون تصميماتك مقصوصة بهذه النسبة تقريباً (أو بنسبة البورتيريه Portrait) حتى تملأ الشاشات بشكل مذهل.
2. **الروابط:** يجب وضع الروابط بين علامتي تنصيص `" "`.
3. لا تحذف مسافات الكود الأساسية عند تغيير النصوص.
